## Описание ##
Это учебный проект. Сборник REST-сервисов.

Решение задачи `1. Синхронный обмен сообщений [#458496]`
#### Сервис core ####
REST-сервис, исполняющий все операции с паспортами.
#### Сервис client ####
REST-сервис, обращающийся к core через RestTemplate.
